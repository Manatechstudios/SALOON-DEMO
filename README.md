# AURUM — Premium Unisex Salon Website

A complete, production-ready static frontend website for a luxury unisex salon.

## Project Structure

```
aurum-salon/
├── index.html      ← Main HTML file
├── style.css       ← All styles (responsive + animations)
├── script.js       ← Interactivity & animations
└── README.md       ← This file
```

## Features

- **Sticky Navbar** with smooth scroll & hamburger menu (mobile)
- **Hero** with animated background image + parallax zoom
- **About** section with overlapping image layout
- **Services** — 29 services across Men's, Women's & Unisex categories
- **Pricing** — 3 transparent price cards
- **Gallery** — 8-image grid with hover zoom & lightbox
- **Testimonials** — 5 dummy client reviews
- **Contact** — WhatsApp, email, map embed, social links
- **Footer** with quick links and social icons
- **Floating WhatsApp button** (bottom-right, always visible)
- **Scroll progress bar** at top of page
- **Scroll-reveal animations** on all sections
- **Service card tilt** on desktop hover

## Customisation

| What to change | Where |
|---|---|
| Salon name | `index.html` → search "AURUM" |
| Phone number | Replace `919876543210` everywhere |
| Address | Contact section in `index.html` |
| Email | Replace `hello@aurumhyd.com` |
| Colors | CSS variables at top of `style.css` |
| Prices | Pricing section in `index.html` |
| Gallery images | Replace Unsplash URLs in `.gallery-item` |

## Opening

Simply open `index.html` in any modern browser — no build step required.
Images load from Unsplash CDN, so an internet connection is needed for images.
